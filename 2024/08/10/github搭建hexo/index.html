github搭建hexo

注册github账号，安装Git，NodeJS

Github：

https://github.com/

Git:

https://git-scm.com/

一路默认即可，然后在桌面打开Git

查询git版本

NodeJS

https://nodejs.org/zh-cn/

一路默认即可，然后打开Git，查询NodeJS和npm版本

安装hexo

打开Git Bash

查询hexo版本

创建hexo源码文件

在合适目录中打开Git Bash

下载完成后里面会有些默认文件。

然后安装hexo需要的依赖

进入hexo-blog目录

回车

安装Github部署插件

在本地查看网页效果

部署到Github

登录Github账号

点击New repository创建仓库

创建index.html主页文件

保存为index.html

访问仓库，可能需要等一小会儿

修改配置文件

用vscode或其他文本工具打开hexo_blog目录下的_config.yml，配置github信息

repo不能以https://开头

repo内容

如果有警告，则要创建一个ssh key

先查看是否存在key，不存在就创建一个

在git窗口中

一直回车，然后获取ssh key公钥

复制内容，在Github右上角的头像的settings中

添加ssh key

验证是否成功

创建token

然后把token添加到_config.yml中

然后就可以部署了

等待一会儿
